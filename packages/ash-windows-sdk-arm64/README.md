# ASH Pattern System Windows SDK

## Platform: ARM64

### Contents
- Headers: windows/src/include/ash/
- Library: ash-core.lib
- PDBs: Debug symbols (if available)

### Installation

1. Extract this ZIP to your target directory
2. Copy ash-core.lib to your project's lib folder
3. Include headers in your project references
4. Link against ash-core.lib

### Build Requirements
- MSVC 18.x (Visual Studio 2022/2026)
- Windows SDK 10.0.x
