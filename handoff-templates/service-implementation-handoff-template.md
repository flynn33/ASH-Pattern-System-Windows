# Service Target Boundary

## Status

Not applicable to the Windows edition.

## Reason

This repository targets Windows desktop systems. It does not define, host, package, test, or release a service or backend runtime.

## Windows Implication

Service hosting, API surfaces, container images, server deployment, request concurrency, service observability, and service secret management are outside the release scope of this repository.

The active target checklist is `desktop-implementation-handoff-template.md`.
