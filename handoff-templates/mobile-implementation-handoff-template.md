# Mobile Target Boundary

## Status

Not applicable to the Windows edition.

## Reason

This repository targets Windows desktop systems. It does not define, build, package, test, or release a mobile application.

## Windows Implication

Mobile lifecycle, app store packaging, device/simulator validation, battery behavior, thermal behavior, and mobile UI requirements are outside the release scope of this repository.

The active target checklist is `desktop-implementation-handoff-template.md`.
